Notifications

The template includes a complete and powerful notification functionality:

  • 6 different positions: left, center and right, each on top or at bottom
  • 25+ options to tweak every aspect of the notifications
  • Similar messages can be grouped to prevent filling the screen
  • The number of simultaneous messages is limited, postponed messages are released as soon as older messages are closed
  • When available (Chrome only currently), the desktop notification API can be used using a single option - fallback to standard notification on other browsers
